UK Grid Reference Widget

This simple widget displays a ten figure UK Ordnance Survey Grid Reference using the same algorithm as the UK Grid Reference data field.  
https://apps.garmin.com/en-GB/apps/2f30fe76-c14a-46e9-a606-76ced27dd776

It is intended to be used for old-school hiking with paper maps when you want to quickly check your current location without the battery drain of running the GPS continuously.

The widget displays the current location and the GPS signal strength.  

The location is displayed using a large font which makes it very easy to read.  

For more information about the meaning of the GPS Accuracy see this forum post: https://forums.garmin.com/archive/index.php/t-167917.html

Note: The widget is primarily intended to display an OS grid ref which only applies to locations in the UK. If you are outside the UK it will display the location as latitude and longitude.

For more information about grid references see: https://www.ordnancesurvey.co.uk/support/the-national-grid.html and https://en.wikipedia.org/wiki/Ordnance_Survey_National_Grid

If you have any feedback on this app please post it to this forum thread: https://forums.garmin.com/showthread.php?305827-DataField-UK-Grid-Reference
